# Contributing

Thanks for wanting to help improve these notes.

## What's welcome

- **Typo / factual corrections** — open a PR directly
- **Better diagrams** — add a PNG/SVG to the chapter's `diagrams/` folder and reference it in the README
- **Additional code examples** — add a notebook to `code/` with a clear filename
- **Missing "Go Deeper" links** — papers, blog posts, tools that pair well with the chapter

## What's not welcome

- Rewriting notes to match a different teaching style — these notes are intentionally written to complement the videos
- Adding frameworks or tools not covered in the videos
- Large refactors to folder structure

## How to contribute

1. Fork the repo
2. Create a branch: `fix/chapter-05-typo` or `add/chapter-09-chroma-example`
3. Make your change — keep it scoped to one chapter
4. Open a pull request with a short description of what changed and why

## Adding a notebook

- Use `code/<descriptive_name>.ipynb` — match the style of existing notebooks
- Include a `pip install` cell at the top
- Include a "Open in Colab" badge in the chapter README
- Test that it runs top-to-bottom on a fresh kernel before submitting

## Questions

Open an issue — it's the fastest way to get a response.
